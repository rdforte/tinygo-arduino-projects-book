# COLOR MIXING LAMP

## Monitor Serial Output

### Flash

```
tinygo flash -target=arduino -port=/dev/tty.usbmodem1401 Project04/main.go
```

### Monitor Serial Output

```
tinygo monitor -baudrate=9600
```
